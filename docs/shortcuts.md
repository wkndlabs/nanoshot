# Shortcuts

Nanoshot is hotkey-driven. Each capture mode has its own global shortcut.

## Defaults

| Mode   | Shortcut |
| ------ | -------- |
| Region | ⌘⇧4      |
| Screen | ⌘⇧3      |
| Window | ⌘⇧5      |

These clash with the macOS built-in screenshot shortcuts until you disable them in **System Settings → Keyboard → Keyboard Shortcuts → Screenshots**. Onboarding links you straight there.

## Customizing

Open **Nanoshot → Settings → Shortcuts** and click any binding to record a new key combination. Nanoshot will warn you if a shortcut conflicts with another macOS or Nanoshot binding.

Bindings are stored per-user and persist across launches.
