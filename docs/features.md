# Features

## Three capture modes

- **Region** — drag to select any rectangle
- **Screen** — full active display
- **Window** — click any window to capture just that window

Each mode has its own global shortcut (see [Shortcuts](/docs/shortcuts)).

## Output

Save captures to:

- The Desktop (default)
- A custom folder of your choice
- The clipboard (no file written)

Configure in **Settings → Capture**.

## Quick preview

After each capture a thumbnail appears in the corner of your choice with **Save** and **Delete** actions. Dismiss it by clicking outside or hitting `Esc`.

The preview corner is configurable in **Settings → Capture → Preview**.

## Desktop cover

Optionally hide desktop icons and wallpaper clutter during captures. Useful when you want a clean background without rearranging your real desktop.

Toggle in **Settings → Capture → Desktop cover**.

## Menubar-first

Nanoshot lives in the menu bar by default. The Dock icon is hidden unless you explicitly enable it in **Settings → General**.

## Launch at login

Enable **Settings → General → Launch at login** to start Nanoshot automatically. Backed by `SMAppService`, so it's a first-class macOS launch agent.
